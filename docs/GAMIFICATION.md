# HaulPass Gamification System

## Philosophy

**Goal**: Reward farmers for using the app consistently and providing quality data that helps the entire network.

**Key Principle**: Gamification should feel natural, not forced. Achievements celebrate real accomplishments and encourage behaviors that improve the platform for everyone.

**Not**: Annoying popups, meaningless badges, or manipulation. Farmers are busyâ€”respect their time.

## Experience Points (XP) System

### How XP Works

**Earn XP by:**
- Completing hauls: +10 XP base
- Entering optional data: +5 XP per field
- Reporting queue: +3 XP per report
- Checking queue before hauling: +2 XP
- Beating your averages: +5 XP
- Helping other farmers (queue reports): +1 XP per farmer helped

**XP Levels:**

| Level | XP Required | Title | Rewards |
|-------|-------------|-------|---------|
| 1 | 0 | Rookie Hauler | Basic features |
| 2 | 100 | Hauler | Unlock insights dashboard |

## Streaks System

**Daily Check-In Streak:**
- Open app and check queue = +1 day
- 7 day streak = +50 XP bonus
- 30 day streak = +200 XP bonus, "Dedicated" badge
- 90 day streak = +1,000 XP bonus, "Committed" badge

**Consecutive Haul Days:**
- Haul 5 days in a row = +100 XP, "Consistent" badge
- Haul 10 days in a row = +300 XP, "Relentless" badge
- Haul 20 days in a row = +1,000 XP, "Unstoppable" badge

**Lost Streak Recovery:**
- Use "Streak Freeze" (earned via achievements)
- 1 free freeze per month
- Premium: 3 freezes per month

## Design Principles

### 1. **Respect Farmers' Time**
- No forced popups during hauls
- Achievements notify AFTER task completion
- Can dismiss/ignore gamification entirely

### 2. **Celebrate Real Accomplishments**
- Achievements = real value delivered
- "You helped 50 farmers" = tangible impact
- "100 loads" = significant seasonal work

### 3. **Reward Quality, Not Quantity**
- Data quality > data volume
- Accurate queue reports > spam reports
- Smart timing > total hauls

### 4. **Make It Optional**
- Can hide gamification UI entirely
- Leaderboards opt-in only
- Social features opt-in only

### 5. **Align with Business Goals**
- Gamification drives data quality
- Data quality improves network value
- Network value attracts more users
- More users = better predictions for everyone
